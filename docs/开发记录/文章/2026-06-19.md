## Markdown 代码块样式调整

- 目标：文章正文中的 Markdown 代码块只保留单层边框，避免 Shiki 高亮容器和 `pre` 包装层形成双框。
- 主要文件：
  - `fluxa-web/src/features/articles/components/markdown-components.tsx`：调整 `pre` 渲染逻辑和普通代码块单框样式。
  - `fluxa-web/src/features/articles/components/markdown-renderer.tsx`：调整 Shiki 代码块容器样式，内部 `pre` 只负责滚动和代码排版。

## Markdown 代码块双框修正

- 目标：修正外层 `pre` 仍然生成边框导致代码块出现双框的问题。
- 主要文件：
  - `fluxa-web/src/features/articles/components/markdown-components.tsx`：将外层 `pre` 改为无边框间距容器，未高亮代码块由 `code` 节点自身提供唯一边框。
  - `fluxa-web/src/features/articles/components/markdown-renderer.tsx`：保留 Shiki 代码块内容容器的唯一边框，内部 `pre` 透明化。
